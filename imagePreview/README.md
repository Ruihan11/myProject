# ZIP Image Viewer

**ZIP Image Viewer** is a Python-based application that enables users to view images inside ZIP files in a slideshow format. The application provides features like auto-play, image scaling, ZIP file management, and fullscreen mode.

## Features

- Automatically detects and extracts images from ZIP files.
- Supports common image formats: `.png`, `.jpg`, `.jpeg`, `.bmp`, `.gif`, `.webp`.
- Navigate images using arrow keys.
- Auto-play slideshow with customizable interval.
- Pause and resume slideshow using `SPACE`.
- Display a list of available ZIP files.
- Load and delete ZIP files directly from the interface or use `DELETE`.
- Toggle fullscreen with the `F11` and exit fullscreen with `ESC`.
- image control `left` and `right`, zip control `enter` and `backspace`.


## Prerequisites

Ensure Python 3.6 or higher is installed. Required libraries: `Pillow` and `tkinter`. Install dependencies using:
```bash
pip install pillow
```
